# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

```shell
curl -fsSL https://raw.githubusercontent.com/TM-MT/LazyVim/main/install.sh | bash
```

Or manually:

```shell
mv ~/.config/nvim{,.bak}
git clone https://github.com/TM-MT/LazyVim.git ~/.config/nvim
```
